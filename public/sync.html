Modo GitHub Pages

Sync local por línea de comandos

La sincronización desde navegador está desactivada en la versión estática. Para actualizar datos, ejecuta el sync local y vuelve a publicar. 

npm run sync:static

Este comando genera/actualiza public/data/config.json y public/data/libraries.json, que son los ficheros usados por GitHub Pages.
